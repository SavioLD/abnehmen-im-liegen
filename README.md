# Abnehmen im Liegen Rottweil – Landingpage

Mobiloptimierte Landingpage für das Studio **Abnehmen im Liegen** in Rottweil
mit integriertem, mehrstufigem Vorqualifizierungs-Formular (Ziel für Meta Ads).

## Inhalt
- `index.html` – die Landingpage
- `css/style.css` – Styling (Weiß & helles Lila)
- `js/app.js` – Formular-Logik & Interaktion
- `assets/` – Fotos & Logo
- `impressum.html`, `datenschutz.html` – Rechtstexte
- `robots.txt`, `sitemap.xml` – SEO

## Lokal ansehen
`index.html` im Browser öffnen.

## Deployment
Statische Seite, kein Build nötig. Einfach bei GitHub Pages, Cloudflare Pages
oder Netlify veröffentlichen (Root = dieses Verzeichnis).

## Formular
Versand via FormSubmit an info@abnehmenimliegen-rottweil.de. Beim ersten
Absenden muss die Adresse einmalig per E-Mail bei FormSubmit bestätigt werden.
